# videoWallpaper-sh

## description

a script to help you set a video wallpaper in linux, only suport x11

## dependent

- ffprobe
- xwinwrap 
- mpv
- bc
- xrandr

## usage

```shell
videoWallpaper.sh [is all monitor use same video]
```

## feature

- [x] auto adapt multi monitor (use xrandr)
- [x] auto configration monitor scale
